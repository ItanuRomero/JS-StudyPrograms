Representando as condicoes em JS

Formato:
if (condicao) {
**comandos executados caso condicao seja verdadeira
}else {
**comandos executados caso condicao seja falsa
}

Tipos de condicao

Condicao simples e condicao composta
a simples apenas executa algo caso condicao seja verdadeira
a composta executa se a condicao for verdadeiraou falsa.
